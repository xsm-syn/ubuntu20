### PERHATIAN
- **Dilarang menjualbelikan script ini**
- Hanya untuk penggunaan personal.
- Status servis terkadang miss informasi. Dimana pada status dead tetapi jika dilihat by servis statusnya sudah aktif. Jadi bisa diabaikan
- Jika mendapatkan error pada status servis dalam jangka panjang, bisa restart servis yang dead.
- **Script ini akan dihapus jika kedapatan ada yang memperjualbelikannya**
- Ada pertanyaan ? Hubungi https://t.me/after_sweet

### INSTALL SCRIPT
<pre><code>bash -c "$(curl -fsSL https://raw.githubusercontent.com/xsm-syn/ubuntu20/main/ins.sh)"</code></pre>

Akses kembali 15 setelah proses instalasi. **Akses kembali via ssh menggunakan port 22**
